# Employee Salary Prediction

## Project Overview

This project predicts employee salary using supervised machine learning techniques.

## Dataset

* File: salary.csv
* Source: https://www.kaggle.com/datasets/ayessa/salary-prediction-classification
* Rows: ~32,000
* Columns: 15
## Problem Statement

The goal of this project is to build machine learning models to predict whether an individual earns more than 50K per year based on demographic and employment-related features. This helps understand factors influencing income levels.

## Task Type

Classification

## Features

Data preprocessing
Model training / analysis
Visualization of results
Evaluation metrics

##Proposed Methodology

Data cleaning (handling missing values)
Encoding categorical variables using Label Encoding
Splitting dataset into training and testing sets
Feature scaling using StandardScaler
Model training and evaluation
Performance comparison using multiple metrics
Cross-validation for robustness

## Machine Learning Models

* Linear Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Random Forest
* Support Vector Machine (SVM)

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1-score
* ROC-AUC

## Results

* Tables: RQ1–RQ7 CSV files
* Visualizations included in notebook
